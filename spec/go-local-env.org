* Overview

Full local development environment for a Go application instrumented with
trace-spine. Everything runs in Docker Compose. Hooks fire from the host
(Claude Code / opencode) against container logs, test output, and telemetry.

** Multi-stack mirroring

The trace-spine exploration focuses on Clojure/ClojureScript as the primary
substrate, but the harness pattern is stack-agnostic. Mirrored versions of
this environment spec exist (or should be created) for each target stack:

| Stack              | App framework          | PBT library          | Debugger           | Status   |
|--------------------+------------------------+----------------------+--------------------+----------|
| Clojure/CLJS       | Ring + Re-frame        | test.check           | CIDER + nREPL      | Primary  |
| Ruby/Rails         | Rails 7.2              | propcheck            | ruby/debug (rdbg)  | Done     |
| Python/FastAPI     | FastAPI + Starlette    | hypothesis           | debugpy            | Planned  |
| TypeScript/Next.js | Next.js + React        | fast-check           | Node --inspect     | Planned  |
| Go/net-http        | net/http + chi         | rapid                | delve (dlv)        | This doc |

The Docker Compose shape is identical across stacks: app + postgres + redis +
kafka + otel-collector + jaeger + mailpit. Only the =web= service image,
debugger port protocol, PBT runner command, and framework-specific hooks change.
The Claude Code harness pattern (Bash + MCP chrome + hooks engine) is unchanged.

// internal/middleware/tracespine.go
package middleware

import (
	"net/http"
	"os"

	tracespine "github.com/aygp-dr/trace-spine/lib/go"
)

// TraceSpine returns middleware that enforces traceparent propagation.
func TraceSpine() func(http.Handler) http.Handler {
	ingress := os.Getenv("TRACE_SPINE_INGRESS") == "true"
	serviceName := os.Getenv("OTEL_SERVICE_NAME")
	if serviceName == "" {
		serviceName = "unknown"
	}

	return tracespine.Middleware(tracespine.Config{
		Ingress:     ingress,
		ServiceName: serviceName,
	})
}

** Initializer: OpenTelemetry

#+begin_src go
// internal/telemetry/otel.go
package telemetry

import (
	"context"
	"log/slog"
	"os"

	"go.opentelemetry.io/otel"
	"go.opentelemetry.io/otel/exporters/otlp/otlptrace/otlptracehttp"
	"go.opentelemetry.io/otel/sdk/resource"
	sdktrace "go.opentelemetry.io/otel/sdk/trace"
	semconv "go.opentelemetry.io/otel/semconv/v1.25.0"
)

// Init configures the global OpenTelemetry trace provider.
func Init(ctx context.Context) (func(), error) {
	serviceName := os.Getenv("OTEL_SERVICE_NAME")
	if serviceName == "" {
		serviceName = "trace-spine-go"
	}

	exporter, err := otlptracehttp.New(ctx)
	if err != nil {
		return nil, err
	}

	res, err := resource.New(ctx,
		resource.WithAttributes(
			semconv.ServiceNameKey.String(serviceName),
		),
	)
	if err != nil {
		return nil, err
	}

	tp := sdktrace.NewTracerProvider(
		sdktrace.WithBatcher(exporter),
		sdktrace.WithResource(res),
	)
	otel.SetTracerProvider(tp)

	shutdown := func() {
		if err := tp.Shutdown(ctx); err != nil {
			slog.Error("shutting down tracer provider", "error", err)
		}
	}

	return shutdown, nil
}

** Structured logging (slog)

#+begin_src go
// internal/logging/logger.go
package logging

import (
	"log/slog"
	"os"

	"go.opentelemetry.io/otel/trace"
)

// Init configures the default slog logger with JSON output.
func Init() {
	handler := slog.NewJSONHandler(os.Stdout, &slog.HandlerOptions{
		Level: slog.LevelInfo,
	})
	slog.SetDefault(slog.New(handler))
}

// WithTrace returns a logger enriched with trace context.
func WithTrace(logger *slog.Logger, span trace.Span) *slog.Logger {
	sc := span.SpanContext()
	return logger.With(
		"trace_id", sc.TraceID().String(),
		"span_id", sc.SpanID().String(),
		"service", os.Getenv("OTEL_SERVICE_NAME"),
	)
}

** Server setup

#+begin_src go
// cmd/server/main.go
package main

import (
	"context"
	"log/slog"
	"net/http"
	"os"
	"os/signal"
	"syscall"
	"time"

	"github.com/go-chi/chi/v5"
	chimiddleware "github.com/go-chi/chi/v5/middleware"
	"go.opentelemetry.io/contrib/instrumentation/net/http/otelhttp"

	"github.com/aygp-dr/trace-spine-app/internal/logging"
	mw "github.com/aygp-dr/trace-spine-app/internal/middleware"
	"github.com/aygp-dr/trace-spine-app/internal/telemetry"
)

func main() {
	logging.Init()

	ctx, cancel := context.WithCancel(context.Background())
	defer cancel()

	shutdownTracer, err := telemetry.Init(ctx)
	if err != nil {
		slog.Error("failed to init telemetry", "error", err)
		os.Exit(1)
	}
	defer shutdownTracer()

	r := chi.NewRouter()

	// Middleware stack: trace-spine first (before request ID)
	r.Use(mw.TraceSpine())
	r.Use(chimiddleware.RequestID)
	r.Use(chimiddleware.RealIP)
	r.Use(chimiddleware.Recoverer)

	// Wrap with OTel HTTP instrumentation
	handler := otelhttp.NewHandler(r, "server",
		otelhttp.WithMessageEvents(otelhttp.ReadEvents, otelhttp.WriteEvents),
	)

	// Health check (outside OTel to avoid noise)
	r.Get("/healthz", func(w http.ResponseWriter, r *http.Request) {
		w.WriteHeader(http.StatusOK)
		w.Write([]byte("ok"))
	})

	// Application routes registered here
	// r.Route("/api", func(r chi.Router) { ... })

	srv := &http.Server{
		Addr:         ":3000",
		Handler:      handler,
		ReadTimeout:  15 * time.Second,
		WriteTimeout: 15 * time.Second,
		IdleTimeout:  60 * time.Second,
	}

	// Graceful shutdown
	go func() {
		sigCh := make(chan os.Signal, 1)
		signal.Notify(sigCh, syscall.SIGINT, syscall.SIGTERM)
		<-sigCh
		slog.Info("shutting down server")
		shutdownCtx, shutdownCancel := context.WithTimeout(ctx, 10*time.Second)
		defer shutdownCancel()
		srv.Shutdown(shutdownCtx)
	}()

	slog.Info("starting server", "addr", srv.Addr)
	if err := srv.ListenAndServe(); err != http.ErrServerClosed {
		slog.Error("server error", "error", err)
		os.Exit(1)
	}
}

*** /loop 5m -- Log scan for invariant violations

#+begin_src bash
#!/usr/bin/env bash
# bin/dx-log-scan-docker -- Scan Go container logs for I-spine-* violations
set -euo pipefail

LOGS=$(docker compose logs web --since 5m --no-log-prefix 2>/dev/null)
VIOLATIONS=0

# I-spine-present: missing traceparent
COUNT=$(echo "$LOGS" | grep -ci "traceparent.*missing\|no.traceparent\|tracespine: missing context" || true)
if [[ "$COUNT" -gt 0 ]]; then
  echo "VIOLATION I-spine-present: $COUNT occurrence(s) in last 5m"
  VIOLATIONS=$((VIOLATIONS + COUNT))
fi

# I-spine-wellformed: malformed header
COUNT=$(echo "$LOGS" | grep -ci "malformed.traceparent\|tracespine: parse error" || true)
if [[ "$COUNT" -gt 0 ]]; then
  echo "VIOLATION I-spine-wellformed: $COUNT occurrence(s)"
  VIOLATIONS=$((VIOLATIONS + COUNT))
fi

# I-spine-no-fabrication: internal origination
COUNT=$(echo "$LOGS" | grep -ci "programmer.error.*originate\|tracespine: fabrication" || true)
if [[ "$COUNT" -gt 0 ]]; then
  echo "VIOLATION I-spine-no-fabrication: $COUNT occurrence(s)"
  VIOLATIONS=$((VIOLATIONS + COUNT))
fi

# I-spine-no-leak: trace ID in response body
COUNT=$(echo "$LOGS" | grep -cE '"trace_id":"[0-9a-f]{32}".*response_body' || true)
if [[ "$COUNT" -gt 0 ]]; then
  echo "VIOLATION I-spine-no-leak: $COUNT occurrence(s)"
  VIOLATIONS=$((VIOLATIONS + COUNT))
fi

# Budget: slow propagation
COUNT=$(echo "$LOGS" | grep -ci "tracespine: budget exceeded\|propagation.*exceeded.*budget" || true)
if [[ "$COUNT" -gt 0 ]]; then
  echo "VIOLATION I-spine-budget: $COUNT occurrence(s)"
  VIOLATIONS=$((VIOLATIONS + COUNT))
fi

# Summary
if [[ $VIOLATIONS -gt 0 ]]; then
  echo "---"
  echo "TOTAL: $VIOLATIONS violation(s) in last 5m"
  echo "View traces: http://localhost:16686"
  exit 1
fi

echo "No violations in last 5m"

*** /loop 30m -- Trace continuity sampling

#+begin_src bash
#!/usr/bin/env bash
# bin/dx-trace-continuity -- Sample recent traces from Jaeger for discontinuity
set -euo pipefail

SERVICE="${OTEL_SERVICE_NAME:-checkout}"
JAEGER="http://localhost:16686"

# Fetch recent traces via Jaeger API
TRACES=$(curl -sf "$JAEGER/api/traces?service=$SERVICE&limit=10&lookback=30m" \
  | jq -r '.data[]' 2>/dev/null)

if [[ -z "$TRACES" ]]; then
  echo "No traces found in last 30m (is the app receiving traffic?)"
  exit 0
fi

# Check each trace for discontinuity
DISCONTINUITIES=0
TRACE_COUNT=$(echo "$TRACES" | jq -s 'length')

for i in $(seq 0 $((TRACE_COUNT - 1))); do
  TRACE=$(echo "$TRACES" | jq -s ".[$i]")
  TRACE_ID=$(echo "$TRACE" | jq -r '.traceID')
  SPAN_COUNT=$(echo "$TRACE" | jq '.spans | length')
  UNIQUE_TRACES=$(echo "$TRACE" | jq '[.spans[].traceID] | unique | length')

  if [[ "$UNIQUE_TRACES" -gt 1 ]]; then
    echo "DISCONTINUITY: trace $TRACE_ID has $UNIQUE_TRACES different trace IDs across $SPAN_COUNT spans"
    DISCONTINUITIES=$((DISCONTINUITIES + 1))
  fi
done

echo "Sampled $TRACE_COUNT traces: $DISCONTINUITIES discontinuity(s)"
if [[ $DISCONTINUITIES -gt 0 ]]; then
  echo "View: $JAEGER/search?service=$SERVICE"
  exit 1
fi

#!/usr/bin/env bash
# bin/dx-migration-check -- Detect pending migrations or schema drift
set -euo pipefail

VERSION=$(docker compose exec -T web migrate \
  -path /app/migrations \
  -database "$DATABASE_URL" \
  version 2>&1)

if echo "$VERSION" | grep -qi "dirty"; then
  echo "DIRTY MIGRATION STATE: database is in a dirty state"
  echo "$VERSION"
  echo "Run: docker compose exec web migrate -path /app/migrations -database \"\$DATABASE_URL\" force VERSION"
  exit 1
fi

# Check for unapplied migration files
LATEST_FILE=$(ls -1 migrations/*.up.sql 2>/dev/null | sort -V | tail -1 | grep -oP '\d+' | head -1)
CURRENT=$(echo "$VERSION" | grep -oP '\d+' || echo "0")

if [[ -n "$LATEST_FILE" ]] && [[ "$LATEST_FILE" -gt "$CURRENT" ]]; then
  echo "PENDING MIGRATIONS: current=$CURRENT, latest file=$LATEST_FILE"
  echo "Run: docker compose exec web migrate -path /app/migrations -database \"\$DATABASE_URL\" up"
  exit 1
fi

echo "Database: migration version $CURRENT, none pending"

** Claude Code + Playwright via MCP chrome

The =claude-in-chrome= MCP server gives Claude Code browser automation
equivalent to Playwright, operating against the live Go app:

*** Trace verification through the browser

#+begin_src text
Scenario: Verify checkout flow carries traceparent end-to-end

1. Claude Code -> mcp__claude-in-chrome__navigate
   URL: http://localhost:3000/products

2. Claude Code -> mcp__claude-in-chrome__find
   Find: "Add to Cart" button -> click

3. Claude Code -> mcp__claude-in-chrome__navigate
   URL: http://localhost:3000/checkout

4. Claude Code -> mcp__claude-in-chrome__read_network_requests
   Filter: XHR to /api/checkout
   Assert: request headers contain "traceparent" matching
           ^00-[0-9a-f]{32}-[0-9a-f]{16}-[0-9a-f]{2}$

5. Claude Code -> Bash
   curl -sf "http://localhost:16686/api/traces?service=checkout&limit=1"
   Assert: trace spans form connected DAG, single trace_id throughout

6. Claude Code -> mcp__claude-in-chrome__read_console_messages
   Pattern: "TraceSpine"
   Assert: no error-level messages
#+end_src

*** DOM inspection for I-spine-no-leak

#+begin_src text
Scenario: Verify trace IDs don't leak to rendered HTML

1. Claude Code -> mcp__claude-in-chrome__navigate
   URL: http://localhost:3000/orders/123

2. Claude Code -> mcp__claude-in-chrome__get_page_text
   Search page text for 32-character hex strings

3. Claude Code -> mcp__claude-in-chrome__javascript_tool

** Debugger integration

Claude Code can attach to the delve debugger when investigating
invariant violations discovered by hooks:

#+begin_src text
Scenario: Hook detects I-spine-no-fabrication, need to inspect middleware

1. Log scan hook reports: "VIOLATION I-spine-no-fabrication"

2. Claude Code reads the violation log to identify the handler/route

3. Claude Code -> Bash
   dlv connect localhost:2345
   # Set a conditional breakpoint at the trace-spine middleware
   # (dlv) break tracespine.Middleware
   # (dlv) condition 1 ctx.Value(tracespine.ContextKey) == nil

4. Developer triggers the request via browser or curl

5. Claude Code -> Bash
   dlv connect localhost:2345
   # Inspect: goroutine, r.Header.Get("Traceparent"), config.Ingress
   # Determine: why the middleware raised a fabrication error

6. Claude Code reports findings and suggests fix
#+end_src

|                                                                     |
