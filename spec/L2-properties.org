* Overview

L2 properties are generative tests run against every adapter implementation.
Failure produces a shrunken counterexample bound to a seed; the seed is the bug report.

* Generators

#+begin_src clojure
;; WellFormedTraceContext generator
(def gen-trace-id
  (gen/fmap #(apply str %)
            (gen/vector (gen/elements "0123456789abcdef") 32)))

(def gen-span-id
  (gen/fmap #(apply str %)
            (gen/vector (gen/elements "0123456789abcdef") 16)))

(def gen-flags
  (gen/elements ["00" "01"]))

(def gen-trace-context
  (gen/let [trace-id gen-trace-id
            span-id  gen-span-id
            flags    gen-flags]
    {:trace-id trace-id
     :span-id  span-id
     :flags    flags}))

;; EmptyCarrier generator
(def gen-empty-carrier
  (gen/return {}))
#+end_src

* Properties

** roundtrip

#+begin_src text
property roundtrip:
  forall ctx :: WellFormedTraceContext,
         carrier :: EmptyCarrier:
    extract(after(inject(ctx, carrier), carrier)) == Some(ctx)
#+end_src

Refutes: I-spine-wellformed (if format/parse is broken)

Seed file: ~conformance/seeds/roundtrip.edn~

** continuation

#+begin_src text
property continuation:
  forall ctx :: WellFormedTraceContext,
         req :: IngressRequest with traceparent = ctx:
    let downstream_calls = simulate_handler(req)
    in forall call in downstream_calls:
         extract(call.carrier).trace_id == ctx.trace_id
#+end_src

Refutes: I-spine-monotonic, C-1

Seed file: ~conformance/seeds/continuation.edn~

** no_fabrication

property no_fabrication:

      result is error AND no trace_id is generated

** no_leak

#+begin_src text
property no_leak:
  forall ctx :: WellFormedTraceContext,
         error :: ServiceError:
    let response = format_error_response(error, ctx)
    in NOT contains(response.body, ctx.trace_id)
       AND NOT contains(response.headers["X-Request-Id"], ctx.trace_id)

* Running the properties

# Replay a failing seed
./run-properties.sh --replay conformance/seeds/roundtrip-failure-12345.edn
#+end_src

* Shrinking

When a property fails, the harness shrinks the counterexample to minimal size:

1. Reduce trace-id to smallest failing value
2. Reduce carrier to minimal failing state
3. Record shrunken example with seed to ~conformance/seeds/~
4. The seed file IS the bug report; attach to issue
