* Overview

This document serves as the central navigation point for all trace-spine documentation.
trace-spine is the wire-discipline library and CI harness that enforces W3C traceparent
propagation across every wire crossing in the Petstore Commerce Platform.

** Key Invariants

| ID | Claim | Refutation |
|----|-------|------------|
| I-spine-present | Every crossing has traceparent | Missing header observed |
| I-spine-wellformed | traceparent matches W3C syntax | Regex validation fails |
| I-spine-monotonic | Trace ID preserved in causal cone | Different trace IDs in same trace |
| I-spine-no-fabrication | Only ingress creates trace IDs | Internal service originates trace |
| I-spine-no-leak | Trace IDs redacted from user-visible output | Trace ID in response/log |
| I-spine-budget | Propagation < 50us p99 | Benchmark exceeds budget |
