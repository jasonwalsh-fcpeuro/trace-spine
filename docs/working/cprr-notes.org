* Overview

CPRR (Conjecture-Prediction-Result-Refutation) is recorded durably in git notes.
Notes are pushed alongside branches, ensuring reasoning history survives rebases,
force-pushes, and repository reconstruction.

* Notes Refs

Five notes refs, all pushed to both origin and archive:

| Ref                      | Purpose                                    |
|--------------------------|--------------------------------------------|
| refs/notes/conjecture    | Hypotheses about system behavior           |
| refs/notes/refutation    | Evidence that falsifies a conjecture       |
| refs/notes/experiment    | Reproducible test runs with environment    |
| refs/notes/deviation     | Intentional departures from spec           |
| refs/notes/test          | Property test seeds and counterexamples    |

* JSON Envelope

Each note is a JSON object with a standard envelope:

#+begin_src json
{
  "schema": "cprr-<dimension>/v1",
  "trace_id": "<ULID from work session>",
  "commit": "<commit SHA this note attaches to>",
  "timestamp": "<ISO 8601 UTC>",
  "<dimension>": { ... }
}
#+end_src

Schemas are defined in ~schema/notes/~.

* CLI Wrappers

** bin/cprr-add

#+begin_src bash
# Add a conjecture note
bin/cprr-add conjecture --commit HEAD \
  --statement "Ring middleware extracts traceparent" \
  --cprr-ref C-1 \
  --refutation "Property test finds rejected valid traceparent"

# Add an experiment note
bin/cprr-add experiment --commit HEAD \
  --seed 42 \
  --command "clj -M:test"

# Add a refutation note
bin/cprr-add refutation --commit HEAD \
  --conjecture-id C-local-001 \
  --evidence "Counterexample found" \
  --counterexample "uppercase hex"
#+end_src

** bin/cprr-show

#+begin_src bash
# Show all notes for a commit
bin/cprr-show HEAD

# Show notes of specific type
bin/cprr-show --type conjecture HEAD

# Show notes across range
bin/cprr-show --type refutation main..HEAD

# Export to NDJSON
bin/cprr-show --format ndjson > cprr-export.ndjson
#+end_src

* Git Commands (Manual)

#+begin_src bash
# Add a note
echo '{"schema":"cprr-conjecture/v1",...}' | git notes --ref=refs/notes/conjecture add -F - HEAD

# Show notes for a commit
git notes --ref=refs/notes/conjecture show HEAD

# List all notes in a ref
git notes --ref=refs/notes/conjecture list

# Push notes
git push origin refs/notes/conjecture

# Fetch notes
git fetch origin refs/notes/conjecture:refs/notes/conjecture
#+end_src

* Integrity Properties

1. Every commit with a Conjecture: trailer SHOULD have a conjecture note
2. Every refutation note MUST reference an existing conjecture
3. Experiment notes MUST have reproducible environment info
4. Deviation notes MUST have a refutation condition

bin/rebuild-check verifies: count(refutation notes reachable from main) ≥
count(CPRR rows in spec/cprr.org with status=refuted).
