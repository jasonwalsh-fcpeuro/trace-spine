* Overview

This document specifies the API Gateway for the Petstore Commerce Platform.
The gateway serves as the unified entry point for all client applications,
providing REST and GraphQL endpoints with integrated trace-spine middleware.

** Design Principles

1. **Trace-first**: Every request carries W3C traceparent, propagated to all downstream services
2. **Security-first**: JWT authentication with feature-flag-based access control
3. **Rate-limited**: Per-user and per-IP rate limiting via Redis
4. **Dual-protocol**: REST for CRUD operations, GraphQL for flexible queries

** Authentication Flow

#+begin_src mermaid :file auth-flow.png :eval never-export
sequenceDiagram
    participant Client
    participant API as API Gateway
    participant Auth as Auth Middleware
    participant Flags as Feature Flags
    participant Service as Backend Service

    Client->>+API: Request (Authorization: Bearer <token>)
    API->>+Auth: Validate JWT
    Auth->>Auth: Verify signature, expiry
    Auth-->>-API: {:user-id "..." :scopes [...]}
    API->>+Flags: Check feature access
    Flags-->>-API: {:allowed true}
    API->>+Service: Forward (traceparent, user context)
    Service-->>-API: Response
    API-->>-Client: Response
#+end_src

* Rate Limiting

** Configuration

| Tier       | Requests/min | Requests/hour | Burst |
|------------+--------------+---------------+-------|
| Anonymous  |           30 |           500 |    10 |
| Registered |          120 |          5000 |    50 |
| Premium    |          600 |         25000 |   200 |
| Internal   |        10000 |           N/A |  1000 |

** Headers

Every response includes rate limit headers:

#+begin_src text

X-RateLimit-Reset: 1715266860
X-RateLimit-Policy: "120;w=60"
#+end_src

* GraphQL Endpoint

** Endpoint

~POST /api/graphql~

** Schema

#+begin_src graphql
type Query {
  # Catalog
  products(
    page: Int = 1
    limit: Int = 20
    category: ID
    search: String
    sort: ProductSort = POPULAR
    priceRange: PriceRangeInput
  ): ProductConnection!

  product(id: ID, slug: String): Product
  categories: [Category!]!
  category(id: ID, slug: String): Category

  # Cart
  cart: Cart

  # Orders
  orders(page: Int = 1, limit: Int = 20): OrderConnection!
  order(id: ID!): Order

  # Wallet
  wallet: Wallet

  # User
  me: User
}

type Mutation {
  # Cart
  addToCart(input: AddToCartInput!): Cart!
  updateCartItem(id: ID!, quantity: Int!): Cart!
  removeFromCart(id: ID!): Cart!
  clearCart: Cart!

  # Checkout
  checkout(input: CheckoutInput!): Order!

  # Orders
  cancelOrder(id: ID!): Order!

  # Wallet
  addGiftCard(code: String!): Wallet!
}

type Subscription {
  orderStatusChanged(orderId: ID!): Order!
}

type Product {
  id: ID!
  name: String!
  slug: String!
  description: String
  price: Money!
  compareAtPrice: Money
  category: Category!
  images: [Image!]!
  variants: [ProductVariant!]!
  inventory: Inventory!
  attributes: JSON
  reviews: ReviewSummary!
  createdAt: DateTime!
  updatedAt: DateTime!
}

type ProductConnection {
  edges: [ProductEdge!]!
  pageInfo: PageInfo!
  totalCount: Int!
}

type ProductEdge {
  node: Product!
  cursor: String!
}

type Category {
  id: ID!
  name: String!
  slug: String!
  description: String
  parent: Category
  children: [Category!]!
  products(page: Int, limit: Int): ProductConnection!
}

type Cart {
  id: ID!
  items: [CartItem!]!
  subtotal: Money!
  estimatedTax: Money!
  estimatedTotal: Money!
  itemCount: Int!
}

type CartItem {
  id: ID!
  product: Product!
  variant: ProductVariant
  quantity: Int!
  unitPrice: Money!
  lineTotal: Money!
}

type Order {
  id: ID!
  number: String!
  status: OrderStatus!
  items: [OrderItem!]!
  shippingAddress: Address!
  billingAddress: Address!
  subtotal: Money!
  shipping: Money!
  tax: Money!
  discount: Money
  total: Money!
  payments: [Payment!]!
  shipments: [Shipment!]!
  createdAt: DateTime!
  updatedAt: DateTime!
}

type Money {
  amount: Float!
  currency: String!
}

enum OrderStatus {
  PENDING
  CONFIRMED
  PROCESSING
  SHIPPED
  DELIVERED
  CANCELLED
  REFUNDED
}

enum ProductSort {
  POPULAR
  PRICE_ASC
  PRICE_DESC
  NEWEST
  NAME_ASC
  NAME_DESC
}

input AddToCartInput {
  productId: ID!
  variantId: ID
  quantity: Int!
}

input CheckoutInput {
  shippingAddress: AddressInput!
  billingAddress: AddressInput
  sameAsBilling: Boolean = false
  paymentMethod: PaymentMethodInput!
  shippingMethod: String!
  promoCode: String
}

scalar DateTime
scalar JSON

* Trace Propagation

** Request Headers

Clients SHOULD include traceparent header:

#+begin_src text
traceparent: 00-4bf92f3577b34da6a3ce929d0e0e4736-00f067aa0ba902b7-01
tracestate: petstore=tenant123
#+end_src

** Behavior

1. If traceparent present: Extract and continue trace, create child span
2. If traceparent absent on ingress: Generate new trace_id and span_id
3. If traceparent absent on internal call: Reject with 400 (programmer error)

** Response Headers

All responses include:

#+begin_src text
traceparent: 00-4bf92f3577b34da6a3ce929d0e0e4736-a1b2c3d4e5f6a7b8-01
X-Trace-Id: 4bf92f3577b34da6a3ce929d0e0e4736
#+end_src

* Error Codes

| Code | HTTP Status | Description                    |
|------+-------------+--------------------------------|
| 400  | Bad Request | Invalid request format         |
| 401  | Unauthorized | Missing or invalid token      |
| 403  | Forbidden   | Insufficient permissions       |
| 404  | Not Found   | Resource not found             |
| 409  | Conflict    | Resource conflict (e.g., cart) |
| 422  | Unprocessable | Validation failed            |
| 429  | Too Many Requests | Rate limit exceeded      |
| 500  | Internal Error | Server error                |
| 502  | Bad Gateway | Downstream service error       |
| 503  | Unavailable | Service temporarily down       |
